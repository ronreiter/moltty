Why Moltty?

💻

Native Performance

Claude Code runs locally on your Mac with full access to your filesystem, credentials, and tools. No containers, no sandboxes.

🌐

Access From Anywhere

Connect from your phone, tablet, or any browser. Your sessions are always running, always accessible.

🔄

Session Persistence

Sessions survive disconnects. Close your laptop, reopen it tomorrow — your Claude sessions auto-resume right where you left off.

📜

Scrollback Buffer

Every session maintains a scrollback buffer. Connect a new viewer and instantly see everything Claude has done.

👥

Multi-Viewer

Multiple clients can watch and interact with the same session simultaneously. Pair program with Claude from different devices.

🔑

Session History

Browse and resume any previous Claude conversation. See the working directory, summary, and size of each session at a glance.

Architecture

A Mac app as the worker, a Go server as the relay, and any client as a viewer.

[Mac App / Viewer] --ws--> [Server: relay + scrollback] <--ws-- [Mac App / Worker] [Web Browser] --ws--> [Server] [Phone] --ws--> [Server] Worker = Mac Electron app, runs claude via node-pty Server = Go relay, stores scrollback, fans out to viewers Viewer = Any client with xterm.js (Electron, web, mobile)
